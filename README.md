# 놀친



#####중앙대학교 아싸를 위한 채팅 어플!#####

##이용 방법##

1. 회원 가입을 한다.(중앙대 이메일 필요 합니다!)
2. 로그인을 한다.
3. 오늘 뭐하고 놀지 정한다.(카테고리 중에서 선택)
4. 원하는 채팅방을 들어가거나, 채팅방을 생성한다.(채팅방 찾기 가능)
5. 채팅방에서 오늘의 놀친을 만나서 재밋게 논다.

----------상대방이 나오지 않을 경우 채팅방에서 상대방 아이디를 클릭 후 '신고하기'를 한다 ----------


##Function##

    1.회원가입 / 회원탈퇴
    2.로그인 / 로그아웃
    3.채팅
    4.채팅방 찾기
    5.신고하기
    6.채팅방
